# Sukku Recon Framework

Sukku is a lightweight recon automation framework built with Go.

## Features

- Subdomain Enumeration
- Live Host Detection
- Port Scanning
- Nuclei Vulnerability Scanning
- JavaScript File Finder

## Tools Used

- Subfinder
- HTTPX
- Nuclei
- Katana
- Nmap

## Installation

```bash
git clone https://github.com/yourusername/sukku.git
cd sukku
go build -o sukku
```

## Usage

```bash
./sukku -d target.com
```

## Help Menu

```bash
./sukku -h
```

## Version

```bash
./sukku -v
```

## Output Files

- subdomains.txt
- live.txt
- ports.txt
- nuclei.txt
- jsfiles.txt
